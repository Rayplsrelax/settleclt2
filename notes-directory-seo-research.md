# Directory Launch Size & SEO Research

## Optimal Directory Size for Launch
- Current: 400+ listings across 40 categories
- Industry consensus: 400+ is already strong for a local city directory
- Key principle: "Quality over quantity" - complete, accurate listings beat half-finished profiles
- For a niche local directory (Charlotte), 400+ with balanced category coverage is launch-ready
- Focus should be on ensuring each listing has complete data (name, category, description, area, phone, website)
- Recommendation: Current 400+ is good for launch. Grow to 500-600 organically through Claim This Business and community submissions

## SEO Implementation Checklist (from Google docs + best practices)

### 1. JSON-LD Structured Data
- Organization schema on homepage
- LocalBusiness schema on directory listing pages (per listing)
- Event schema on events pages
- Article/BlogPosting schema on blog posts
- BreadcrumbList on all pages
- WebSite with SearchAction (already exists in index.html)

### 2. LocalBusiness Required Properties
- name, address (streetAddress, addressLocality, addressRegion, postalCode, addressCountry)
- Recommended: geo (lat/lng), telephone, url, priceRange, openingHoursSpecification

### 3. Event Schema Properties
- name, startDate, endDate, location (name, address), description, url, image

### 4. Meta Tags Strategy
- Unique title + description per page
- Open Graph tags (og:title, og:description, og:image, og:url)
- Twitter card tags
- Canonical URLs

### 5. Sitemap Improvements
- Already has static pages + neighborhoods + blog slugs
- Need to add: individual directory category pages, event detail URLs if applicable
- Ensure lastmod dates are accurate

### 6. Technical SEO
- robots.txt (already exists)
- Semantic HTML (h1-h6 hierarchy, alt text, aria labels)
- Page speed optimization
- Mobile-first design (already responsive)
